Nike Power BI Sales Dashboard

 About the Project
This project presents an interactive Power BI dashboard analyzing Nike US sales performance.
The dashboard focuses on sales trends, top-performing products, regional performance,
and growth comparison between 2020 and 2021.

 Key Insights
- Significant sales growth from 2020 to 2021
- West region achieved the highest total sales
- Nike Air Max models dominate top product sales
- Online sales show strong contribution compared to other channels
